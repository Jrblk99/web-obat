# Display Obat - Fullscreen Otomatis

Aplikasi web untuk menampilkan daftar obat secara otomatis di layar penuh.

## Fitur
- Rotasi otomatis setiap 5 detik
- Upload foto PNG/JPG
- Data disimpan di browser (localStorage)
- Tidak butuh server — jalan di GitHub Pages!

## Cara Deploy ke GitHub Pages
1. Buat repositori GitHub
2. Upload semua file ke repositori
3. Settings → Pages → Pilih branch `main` → Save
4. Buka: `https://username.github.io/nama-repo/`
